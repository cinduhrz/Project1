# Life Advice Generator

An app to search quotes by keyword.
Bonus functions to be added include the ability to search a quote by author and to randomly generate a quote.

### Technologies Used

- HTML
- CSS
- Javascript
- jQuery
- ZenQuotes API
- Mural
- VS Code
- Vercel
